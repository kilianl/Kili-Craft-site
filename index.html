Kili-Craft

A Minecraft Server

Our server is special, We try to make you comfortable when playing on the server! Our server is mainly roleplay, but if you don't want to take part in this you can also survive with some friends. PvP is allowed but not to be abused.

This site is still in progress.

About the server

We offer a great amount of variety to the game with the help of usefull plugins. We also have a dedicated staff trying to make your stay as pleasant as we can, if you have any suggestions, complaints or just random thoughts; you can post those on ours forums. We are always glad when we receive feedback from our players.
